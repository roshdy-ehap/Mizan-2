# 📱 Mizan POS - نظام نقاط البيع

نظام محاسبي احترافي لإدارة المتاجر والمحلات - مصنوع بـ **Electron** و **HTML5**.

## ✨ الميزات الرئيسية:

- ✅ **لوحة تحكم ديناميكية** - إحصائيات فوري وتقارير شاملة
- ✅ **إدارة الفواتير** - إنشاء وحفظ ومتابعة الفواتير
- ✅ **إدارة المنتجات** - قائمة منتجات شاملة مع أسعار وكميات
- ✅ **إدارة العملاء** - حفظ بيانات العملاء والمشتريات السابقة
- ✅ **النسخ الاحتياطية** - حفظ واستعادة البيانات
- ✅ **التقارير** - تحليل المبيعات والإحصائيات
- ✅ **الواجهة العربية** - دعم كامل للعربية RTL
- ✅ **لا يحتاج انترنت** - يعمل بدون اتصال بالانترنت
- ✅ **آمن تماماً** - البيانات محفوظة محليّاً على الجهاز

## 🚀 البدء السريع:

### المتطلبات:

- **Windows 7** أو أحدث
- **Node.js 16+** (للتطوير فقط)

### التثبيت والتشغيل:

#### الطريقة 1: تشغيل مباشر (الأسهل) ⭐

```bash
# 1. حمّل الملف
Mizan-POS-3.1.0.exe

# 2. شغّله مباشرة - بدون تثبيت
double-click Mizan-POS-3.1.0.exe

# 3. ابدأ الاستخدام
```

**لا تحتاج:** تثبيت Python، Visual C++، أو أي متطلبات أخرى.

#### الطريقة 2: البناء من المصدر (للمطورين)

```bash
# 1. انسخ المشروع
git clone https://github.com/roshdy-ehap/Mizan-2.git
cd Mizan-2

# 2. ثبت المكتبات
npm install

# 3. شغّل في وضع التطوير
npm start

# 4. أو ابن الـ EXE
npm run build
```

#### الطريقة 3: استخدام ملفات Batch (Windows)

```bash
# للتطوير:
double-click run_dev.bat

# للبناء:
double-click build.bat
```

## ⚙️ نظام البناء:

### البناء المحلي:

```bash
# طريقة 1: استخدام npm
npm run build

# طريقة 2: استخدام build.bat
build.bat

# طريقة 3: مع caching
npm run build:cached
```

### البناء التلقائي:

يتم البناء **تلقائياً** عند كل `push` على GitHub Actions:

1. **الدفع:** `git push`
2. **البناء:** GitHub Actions يبني الـ EXE
3. **التحميل:** اذهب إلى **Actions** → **Artifacts**
4. **التحميل:** حمّل `Mizan-POS-3.1.0.exe`

## 🐛 حل المشاكل الشائعة:

### ❌ الخطأ: `ERR_FILE_NOT_FOUND`

**الحل:**
```bash
# تأكد أن src/index.html موجود
ls src/index.html

# أعد بناء
npm run build
```

### ❌ الخطأ: `node: command not found`

**الحل:**
1. حمّل Node.js من https://nodejs.org
2. اختر **LTS**
3. ثبت البرنامج
4. أعد تشغيل build.bat

### ❌ الخطأ: `npm install` بطيء جداً

**الحل:**
```bash
# استخدم mirror عربي
npm config set registry https://registry.npmmirror.com

# ثم أعد المحاولة
npm install
```

### ❌ المشروع بطيء في البناء

**الحل:**
```bash
# امسح المكتبات القديمة
rm -r node_modules

# أعد التثبيت
npm install

# أعد البناء
npm run build
```

### ❌ الـ EXE كبير جداً (200 MB+)

**الحل:**
```bash
# امسح مجلد dist
rm -r dist

# ابن من جديد
npm run build

# استخدم Portable بدلاً من Installer
npm run build:portable
```

## 📚 الأسئلة الشائعة:

### س: هل يحتاج تثبيت؟
**ج:** لا - الملف `.exe` يشتغل مباشرة بدون تثبيت.

### س: هل يحتاج انترنت؟
**ج:** لا - يعمل بالكامل بدون انترنت. البيانات محفوظة محليّاً.

### س: هل البيانات آمنة؟
**ج:** نعم - البيانات محفوظة محليّاً على جهازك فقط.

### س: كيف أنقل البيانات لجهاز آخر؟
**ج:** استخدم **النسخ الاحتياطية** → احفظ الملف → استعده في الجهاز الجديد.

### س: هل يعمل على Mac/Linux؟
**ج:** الملف الحالي لـ Windows فقط. يمكن بناء نسخ أخرى بطلب.

### س: هل يمكن إضافة ميزات جديدة؟
