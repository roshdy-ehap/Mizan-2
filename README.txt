         بدون Python — بدون تثبيت — EXE واحد

╔═══════════════════════════════════════════╗
║       كيف تبني الـ EXE                   ║
╚═══════════════════════════════════════════╝

المتطلب الوحيد (مرة واحدة على جهازك فقط):
  ← Node.js من https://nodejs.org  (اختر LTS)

بعد تثبيت Node.js:
  1. افتح مجلد المشروع
  2. شغّل: build.bat
  3. انتظر 5-8 دقائق
  4. الملف الناتج في: dist\ميزان-POS-v3.1.0.exe

انقل الـ EXE لأي جهاز Windows وشغّله — خلاص!

╔═══════════════════════════════════════════╗
║       اختبار بدون بناء                   ║
╚═══════════════════════════════════════════╝

شغّل: run_dev.bat
(محتاج Node.js)

╔═══════════════════════════════════════════╗
║       أين تُحفظ بيانات العميل؟           ║
╚═══════════════════════════════════════════╝

بيانات IndexedDB تُحفظ تلقائياً في:
  C:\Users\[اسم المستخدم]\AppData\Roaming\mizan-pos\

مجلد النسخ الاحتياطية:
  جنب الـ EXE في مجلد "ميزان-نسخ احتياطية"
